Outros seletores

  • * - todo o HTML
  • tag - somente a tag selecionada será alterada
  • .tag - alterar classe de nome tag
  • #tag - alterar id de nome tag
  • tag > tag filha - alterar a tag filha da tag

Pseudo-class

  • tag:hover - muda a cor quando passa o mouse
  • tag:first-child - primeiro nó da tag
  • tag:last-child - último nó da tag
  • tag:nth-child(2) - nó 2 da tag
